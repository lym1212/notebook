## 本质是在当前作用域注入代码

- 接收一个字符串作为参数，并当做语句执行

- 参数字符串无法当作语句运行，就会报错

- 参数字符串，应该有独自存在的意义

  ```javascript
  eval('return;')   // return 必须在函数中使用
  ```

- 参数不是字符串，会原样返回

- 没有自己的作用域，都在当前作用域内执行

  > 如果使用严格模式，`eval`内部声明的变量，不会影响到外部作用域
  >
  > 但是依然可以读写当前作用域的变量

- 别名：`var a = eval`

  > 使用别名执行`eval`时，`eval`内部一律是全局作用域
